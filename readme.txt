== Description ==

Add a subscribe form for Clang on your page without using iFrames

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[myClangForm]` in your templates
